MaixPy Guide
=================================================

MaixPy is an open source MicroPython project based on the Sipeed M1 platform. It supports a variety of peripherals on the Sipeed M1. It is designed to make programming easier. This chapter will detail how to use the MaixPy API.

.. figure:: http://pgeza64pd.bkt.clouddn.com/dan-board.jpg
  :width: 500px
  :align: center
  
  Sipeed M1
 

Please read about the information on `Sipeed M1  <http://dan.lichee.pro/index.html>`_.

For introduction and download of MaixPy, please go to `MaixPy-github <https://github.com/sipeed/MaixPy>`_ and read README.md. 

If you have any questions or suggestions about MaixPy, please go to the `MaixPy bbs <http://bbs.lichee.pro/t/sipeed-m1>`_ to post suggestions and discuss with other developers.

Foreword
----------

This article will assume that you have read the sdk introduction on github and Lichee Dan's wiki, and already have some python knowledge.But we will still try our best to explain the details of the MaixPy API.
MaixPy's API will continue to be updated, please pay close attention to our github or this tutorial for the latest API introduction. This tutorial will introduce the machine module, uos module and apps module related to the Sipeed M1 platform.
